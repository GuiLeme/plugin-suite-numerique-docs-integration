# BlockNote-demo-server for POC in BigBlueButton

This is a demo server for the BigBlueButton integration with `Blocknote` based on the official demo server implementation - [TypeCellOS/BlockNote-demo-nextjs-hocuspocus](https://github.com/TypeCellOS/BlockNote-demo-nextjs-hocuspocus)

## Setup

Just run the `./deploy.sh` script from this server folder.
